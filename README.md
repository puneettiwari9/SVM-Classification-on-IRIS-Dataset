# SVM Classification Project

This repository contains an implementation of Support Vector Machine (SVM) classification using Python. The project demonstrates how to train an SVM model, visualize decision boundaries, analyze results using a confusion matrix, and perform data visualization on the dataset.


##  Project Overview

This project applies an SVM classifier on the dataset you provided. The workflow includes:

* Loading and preprocessing the dataset
* Visualizing data (scatter plots of selected features)
* Training an SVM classifier
* Plotting the SVM decision boundary
* Evaluating the model with a confusion matrix

* Generate plots
